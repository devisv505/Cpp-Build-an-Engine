Portable UDP Datagrams

TCP gives us a reliable byte stream; real-time state updates usually need UDP. We will send individual datagrams and configure the receiver as non-blocking so the game loop never waits for the network. The code uses the same portable layer on Windows and macOS.

What you'll learn
  • Send one datagram with sendto.
  • Bind and receive datagrams with recvfrom.
  • Use portable non-blocking mode and error checks.
  • Drain every waiting packet once per frame.

Sending a datagram

#include "SocketPlatform.h"
#include <string>

bool SendMessage(SocketHandle socket, const char* ip, uint16_t port,
                 const std::string& message) {
    sockaddr_in destination{};
    destination.sin_family = AF_INET;
    destination.sin_port = htons(port);
    if (inet_pton(AF_INET, ip, &destination.sin_addr) != 1) return false;

    const int sent = sendto(socket, message.data(),
        static_cast<int>(message.size()), 0,
        reinterpret_cast<sockaddr*>(&destination), sizeof(destination));
    return sent == static_cast<int>(message.size());
}

A non-blocking receiver

#include "SocketPlatform.h"
#include <iostream>

SocketHandle CreateReceiver(uint16_t port) {
    SocketHandle socket = ::socket(AF_INET, SOCK_DGRAM, IPPROTO_UDP);
    if (socket == InvalidSocket) return InvalidSocket;

    sockaddr_in local{};
    local.sin_family = AF_INET;
    local.sin_addr.s_addr = htonl(INADDR_ANY);
    local.sin_port = htons(port);

    if (bind(socket, reinterpret_cast<sockaddr*>(&local), sizeof(local)) != 0 ||
        !SetNonBlocking(socket)) {
        CloseSocket(socket);
        return InvalidSocket;
    }
    return socket;
}

void DrainPackets(SocketHandle socket) {
    for (;;) {
        char data[1200]{};
        sockaddr_in sender{};
        SocketLength senderLength = sizeof(sender);

        const int received = recvfrom(socket, data, sizeof(data), 0,
            reinterpret_cast<sockaddr*>(&sender), &senderLength);

        if (received < 0) {
            const int error = LastSocketError();
            if (!WouldBlock(error)) std::cerr << "recvfrom failed: " << error << '\n';
            break;
        }

        HandlePacket(data, static_cast<std::size_t>(received), sender);
    }
}

DrainPackets loops because several UDP packets can arrive between rendered frames. The normal stopping condition is “would block,” which means the receive queue is empty right now.

Use it in the engine loop

SocketRuntime runtime;
SocketHandle receiver = CreateReceiver(7777);

while (!window.ShouldClose()) {
    window.PollEvents();
    DrainPackets(receiver);  // returns immediately when the queue is empty
    scene.Update(deltaTime);
    renderer.DrawScene(scene);
}

CloseSocket(receiver);
Stay below the path MTU

Keep real-time UDP payloads around 1200 bytes or less unless you have measured the path. Smaller datagrams reduce IP fragmentation risk.

Recap
  • UDP sends independent datagrams with no connection handshake.
  • The receiver binds to a local port and uses non-blocking mode.
  • The wrapper maps non-blocking setup and “would block” errors across Windows and macOS.
  • Drain the receive queue every frame and validate each packet before using it.
